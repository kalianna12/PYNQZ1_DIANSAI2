# LTC2208 双通道 ADC 2×20Pin 接口定义

## 接口说明

-   接口类型：2×20 Pin，2.54 mm
-   适用于图示 LTC2208 转接板接口
-   奇数引脚位于右侧（1、3、5...39）
-   偶数引脚位于左侧（2、4、6...40）

------------------------------------------------------------------------
